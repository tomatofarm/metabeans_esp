## 2. StatusBadge 컴포넌트

`src/components/common/StatusBadge.tsx` 파일이 없거나 스타일이 다르면 아래 기준으로 만들거나 수정:

```
props: status ('success' | 'warning' | 'danger' | 'info' | 'default'), label: string

스타일 (pill 형태):
  padding: 4px 12px
  border-radius: 20px
  font-size: 0.75rem
  font-weight: 600
  display: inline-flex, align-items: center, gap: 4px

success: background var(--color-success-bg), color var(--color-success-text)
warning: background var(--color-warning-bg), color var(--color-warning-text)
danger:  background var(--color-danger-bg),  color var(--color-danger-text)
info:    background var(--color-info-bg),    color var(--color-info-text)
default: background #F1F5F9, color var(--color-mid)
```

전체 프로젝트에서 Ant Design `<Tag>`, `<Badge>` 등으로 상태를 표시하는 곳을
이 `<StatusBadge>` 컴포넌트로 교체해줘. (기능 로직은 건드리지 말 것)

## 3. Header 컴포넌트

```
height: 64px
background: white
border-bottom: 1px solid var(--color-border)
padding: 0 24px
position: fixed, top:0, left:0, right:0, z-index:1000
display: flex, justify-content: space-between, align-items: center

로고: font-size 1.25rem, font-weight 700, color var(--color-primary)
우측: 알람 아이콘 + 사용자명 + 로그아웃 버튼
```

긴급 알람 아이콘 (Red 알람 존재 시):
```css
/* shake 애니메이션 */
@keyframes shake {
  0%,100% { transform: rotate(0deg); }
  25%      { transform: rotate(-10deg); }
  75%      { transform: rotate(10deg); }
}
/* 알람 배지: position absolute, background var(--color-danger), color white */
/* 폰트 크기 0.65rem, border-radius 9px, min-width 18px, height 18px */
```
