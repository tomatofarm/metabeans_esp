## 2. 장비정보 탭 (EquipmentInfoPage)

### 장비 기본정보 카드
```
카드: background white, border-radius 12px
      padding 20px, box-shadow var(--card-shadow), margin-bottom 20px

카드 제목:
  font-size 1rem, font-weight 700, color var(--color-dark)
  margin-bottom 16px, display flex, align-items center, gap 8px
  아이콘: color var(--color-primary)

정보 그리드:
  display grid, grid-template-columns repeat(2, 1fr), gap 16px

각 정보 항목:
  display flex, flex-direction column, gap 6px

  라벨: font-size 0.8rem, color var(--color-mid), font-weight 500
  값:   padding 10px 14px, background var(--color-bg)
        border-radius 8px, font-size 0.9rem, color var(--color-dark)
        border 1px solid var(--color-border)
```

### 센서 현황 상태 카드 (status-grid)
집진기별 주요 상태 수치를 카드 형태로 표시:

```
레이아웃: display grid, grid-template-columns repeat(3, 1fr), gap 16px

각 상태 카드:
  border-radius 12px, padding 20px, text-align center

  정상:  background linear-gradient(135deg, #D1FAE5, #A7F3D0)
  주의:  background linear-gradient(135deg, #FEF3C7, #FDE68A)
  위험:  background linear-gradient(135deg, #FEE2E2, #FECACA)

  라벨: font-size 0.8rem, color var(--color-mid), margin-bottom 8px
  수치: font-size 1.5rem, font-weight 700, color var(--color-dark)
  상태: font-size 0.75rem, margin-top 4px
        정상 color var(--color-success)
        주의 color var(--color-warning)
        위험 color var(--color-danger)
```

### 파워팩(컨트롤러) 목록
```
각 파워팩 행:
  display flex, align-items center, gap 12px
  padding 12px 16px, border-radius 8px
  background var(--color-bg), margin-bottom 8px

  번호 배지: 24x24, border-radius 50%
             background var(--color-primary), color white
             font-size 0.75rem, font-weight 700

  이름: font-weight 600
  상태: StatusBadge 컴포넌트 (정상/고장/비활성)
```

## 3. 실시간 모니터링 탭 (RealtimeMonitorPage)

### 집진기별 상태 카드 그리드 (sensor-grid)
```
레이아웃: display grid, grid-template-columns repeat(4, 1fr), gap 16px
          margin-bottom 20px

각 센서 카드:
  border-radius 12px, padding 20px, text-align center

  정상:  linear-gradient(135deg, #D1FAE5, #A7F3D0)
  주의:  linear-gradient(135deg, #FEF3C7, #FDE68A)
  위험:  linear-gradient(135deg, #FEE2E2, #FECACA)

  아이콘: font-size 1.5rem, margin-bottom 8px
  항목명: font-size 0.8rem, color var(--color-mid), margin-bottom 4px
  수치:   font-size 1.4rem, font-weight 700, color var(--color-dark)
  단위:   font-size 0.75rem, color var(--color-light)
  상태:   StatusBadge 하단
```

표시 항목 (집진기 단위):
- 연결 상태, 전원 상태, 유입 온도, 필터 점검 상태, PM2.5, PM10

### 연결 상태 표시
```
연결됨:
  dot (8x8, border-radius 50%, background var(--color-success))
  + pulse 애니메이션
  텍스트: "ONLINE", color var(--color-success)

끊김:
  dot background var(--color-danger)
  텍스트: "OFFLINE", color var(--color-danger)
  + 마지막 통신 시각 표시
```

### 상태 변경 알림 메시지 (필터 점검 필요)
```
background #FEF3C7, border-left 4px solid var(--color-warning)
border-radius 8px, padding 16px 20px, margin-bottom 16px

아이콘: ⚠️ 또는 warning 아이콘, color var(--color-warning)
텍스트: font-size 0.9rem, color #92400E
```

### 실시간 차트 (ECharts)
```

  background white, border-radius 12px
  padding 20px, box-shadow var(--card-shadow)

dataZoom (스크롤 확대/축소):
```
