## 1. 장치 제어 탭 (ControlPage)

### 제어 모드 선택 버튼 그룹
자동/수동 모드 전환 버튼:
```
display flex, gap 8px, margin-bottom 24px

각 버튼:
  padding 10px 24px, border-radius 8px
  font-weight 600, cursor pointer, transition all 0.2s
  border 1.5px solid var(--color-border)

  비선택: background white, color var(--color-mid)
  선택됨: background var(--color-primary), color white, border-color transparent
  disabled(안전오버라이드): background #F1F5F9, color var(--color-light)
                            cursor not-allowed, opacity 0.6
```

안전 오버라이드 경고 박스:
```
background #FEE2E2, border-left 4px solid var(--color-danger)
border-radius 8px, padding 16px 20px, margin-bottom 16px
display flex, gap 12px
아이콘: 🚨, 텍스트: font-size 0.9rem, color #991B1B
```

### 전원 제어 카드
```
카드: background white, border-radius 12px, padding 20px

전원 ON/OFF 토글:
  현재 상태 표시: 큰 원형 버튼 (80x80)
  ON:  background var(--color-success), color white, box-shadow 그린 글로우
  OFF: background #E2E8F0, color var(--color-mid)
  transition all 0.3s

텍스트: font-size 1rem, font-weight 600, margin-top 12px
```

### 댐퍼 제어 카드 (8단계)
```
카드 제목 + 현재 개도율 수치 (font-size 2rem, font-weight 700, color var(--color-primary))

8단계 버튼 그리드:
  display grid, grid-template-columns repeat(4, 1fr), gap 8px

각 단계 버튼:
  padding 12px 8px, border-radius 8px, text-align center
  border 1.5px solid var(--color-border), cursor pointer
  transition all 0.2s

  비선택: background white, color var(--color-mid)
  선택됨: background var(--color-primary), color white, border-color transparent
          transform scale(1.05)

  버튼 내용:
    단계 숫자 (font-size 1.1rem, font-weight 700)
    % 수치   (font-size 0.75rem, color 상속)

자동 모드 시: 목표 풍량(CMH) 입력 필드 추가
  input: padding 10px 14px, border 1.5px solid var(--color-border)
         border-radius 8px, font-size 0.95rem
         focus: border-color var(--color-primary), 파란 글로우
```

### 시로코팬 제어 카드
```
팬 속도: OFF / LOW / MID / HIGH (4단계 버튼 그룹, 댐퍼 동일 스타일)
자동 모드 시: 목표 풍속(m/s) 입력 필드
```

### 제어 이력 테이블 (자동 제어 이력)
```
테이블: UI_05 A/S 관리와 동일한 테이블 스타일 적용
컬럼: 시각, 제어 유형, 변경값, 실행 주체, 결과
```

## 2. 이력 조회 탭 (HistoryPage)

### 기간 필터 바
```
display flex, align-items center, gap 12px
background white, padding 16px 20px
border-radius 12px, box-shadow var(--card-shadow), margin-bottom 20px

기간 선택 버튼: 오늘 / 1주 / 1개월 / 3개월 / 직접입력
  각 버튼: padding 8px 16px, border-radius 6px
  비선택: background var(--color-bg), color var(--color-mid)
  선택:   background var(--color-primary), color white

날짜 직접입력 (DatePicker): border 1.5px solid var(--color-border), border-radius 8px
```

### 이력 탭 (서브탭)
```
border-bottom 2px solid var(--color-border), margin-bottom 20px

각 서브탭:
  padding 12px 4px, margin-right 24px
  color var(--color-mid), cursor pointer
  border-bottom 2px solid transparent, margin-bottom -2px
  transition all 0.2s

  active: color var(--color-primary), border-bottom-color var(--color-primary)
```

서브탭 구성: A/S 이력 / 청소 상태 이력 / 화재감지 제어 이력 / 전원장치 제어 이력

### 이력 테이블
```
테이블 컨테이너:
  background white, border-radius 12px
  padding 20px, box-shadow var(--card-shadow)

th: background var(--color-bg), padding 14px 16px
    font-size 0.85rem, color var(--color-mid), font-weight 600
    border-bottom 2px solid var(--color-border)

td: padding 14px 16px, border-bottom 1px solid #F1F5F9
    font-size 0.9rem, color #475569

tr hover: background var(--color-bg), cursor pointer
```
