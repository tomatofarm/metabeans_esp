## 2. 필터 바

```

background white, padding 16px 20px
border-radius 12px, box-shadow var(--card-shadow), margin-bottom 20px

구성 요소:
  검색 인풋 (매장명/사업자번호)
  상태 드롭다운: 전체 / 활성 / 비활성 / 승인대기
  본사 드롭다운 (ADMIN용)
  지역 드롭다운
  초기화 버튼 (secondary 스타일)

드롭다운 스타일:
  padding 10px 14px, border 1.5px solid var(--color-border)
  border-radius 8px, background white, min-width 120px

```

## 4. 고객 목록 테이블

```
테이블 컨테이너:
  background white, border-radius 12px
  padding 0, box-shadow var(--card-shadow), overflow hidden

테이블 헤더:
  background var(--color-bg), padding 14px 20px
  display flex, justify-content space-between, align-items center
  border-bottom 1px solid var(--color-border)

  제목: font-size 1rem, font-weight 700
  건수 배지: background var(--color-bg-hover), color var(--color-primary)
             padding 4px 10px, border-radius 20px, font-size 0.8rem

테이블 th:
  background var(--color-bg), padding 14px 20px
  font-size 0.85rem, color var(--color-mid), font-weight 600
  border-bottom 2px solid var(--color-border), text-align left

테이블 td:
  padding 14px 20px, border-bottom 1px solid #F1F5F9
  font-size 0.9rem, color #475569

tr hover: background var(--color-bg), cursor pointer
```

### 상태 배지
```
활성:    background var(--color-success-bg), color var(--color-success-text)
비활성:  background #F1F5F9, color var(--color-mid)
승인대기: background var(--color-warning-bg), color var(--color-warning-text)
pill 형태 (border-radius 20px, padding 4px 12px)
```

### 액션 버튼 (행 우측)
```

           background var(--color-bg-hover), color var(--color-primary)
           border none, font-size 0.8rem, font-weight 600
```
