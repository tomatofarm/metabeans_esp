## 2. IAQ 현황 패널 (IAQPanel)

현재 테이블/리스트 형태라면 아래 그리드 카드 형태로 교체:

```
패널 헤더:
  "매장명 > 층 실내 공기질 현황" 형식 제목
  우측: [LIVE] 배지 (pulse 애니메이션) + 마지막 업데이트 시각

그리드:
  display grid
  grid-template-columns repeat(5, 1fr)  ← 10개 항목 2줄
  gap 16px

각 IAQ 항목 카드:
  background linear-gradient(135deg, #F8FAFC, #EEF2FF)
  border-radius 12px, padding 20px
  text-align center, border 1px solid var(--color-border)
  transition transform 0.2s, box-shadow 0.2s
  hover: translateY(-2px), box-shadow 강화

  [상] 아이콘 박스 (48x48, border-radius 12px):
    온도:   linear-gradient(135deg, #FF6B6B, #FF8E8E)
    습도:   linear-gradient(135deg, #4ECDC4, #6EE7DE)
    PM2.5:  linear-gradient(135deg, #F59E0B, #FBBF24)
    PM10:   linear-gradient(135deg, #F97316, #FB923C)

  [중] 항목명: font-size 0.85rem, color var(--color-mid), margin-bottom 8px
  [중] 수치:   font-size 1.8rem, font-weight 700, color var(--color-dark)
       단위:   font-size 0.8rem, color var(--color-light), margin-left 4px
```

## 3. 문제 발생 이슈 패널 (IssueListCard)

```
패널 헤더: "문제 발생 이슈" 제목 + 전체 건수 배지

카테고리별 접힘/펼침 섹션 (4개):
  헤더: 카테고리명, Yellow/Red 배지(건수), 펼침 화살표
  배경: 카테고리별 연한 색상 (주의 #FEF3C7, 위험 #FEE2E2)

각 이슈 항목:
  display flex, align-items flex-start, gap 12px
  padding 14px, background var(--color-bg)
  border-radius 10px, margin-bottom 8px

  [좌] 상태 dot: 12x12, border-radius 50%
       Yellow: background var(--color-warning)
       Red:    background var(--color-danger)

  [중] 매장명: font-size 0.75rem, color var(--color-primary), font-weight 600
       집진기명 + 수치: font-size 0.85rem, font-weight 600
       발생 시각: font-size 0.8rem, color var(--color-mid)

  [우] StatusBadge: 주의/위험
```
