## 2. 회원가입 페이지 (4종 공통)

레퍼런스: `docs/ESP_매장점주_회원가입.html`, `docs/ESP_본사_회원가입.html` 등을 읽을 것.

### 단계 표시 (Step Indicator)
```
display flex, justify-content center, gap 0
margin-bottom 40px

각 단계:
  원형 번호 (36x36, border-radius 50%)
  완료:  background var(--color-primary), color white
  현재:  background var(--color-primary), color white, box-shadow 글로우
  미완료: background #E2E8F0, color var(--color-mid)

연결선:
  height 2px, flex 1, margin-top 18px
  완료 구간: background var(--color-primary)
  미완료 구간: background var(--color-border)

단계명 라벨: font-size 0.75rem, margin-top 8px, text-align center
```

### 폼 카드
```
background white, border-radius 16px
padding 32px, box-shadow var(--card-shadow)
max-width 640px, margin 0 auto
```

### 폼 요소 (로그인과 동일 기준)
- 인풋, 라벨 → 로그인과 동일한 스타일 적용
- 필수 항목 표시: 라벨 우측에 빨간 * (color var(--color-danger))

### 파일 업로드 (사업자등록증)
```
border: 2px dashed var(--color-border)
border-radius 12px, padding 40px 20px
text-align center, cursor pointer
transition border-color 0.2s
hover: border-color var(--color-primary), background var(--color-bg-hover)
```

### 하단 버튼 영역
```
display flex, gap 12px, justify-content flex-end, margin-top 32px

이전 버튼: background white, border 1.5px solid var(--color-border)
           color var(--color-mid), padding 12px 24px, border-radius 8px

               padding 12px 24px, border-radius 8px, font-weight 600
```
