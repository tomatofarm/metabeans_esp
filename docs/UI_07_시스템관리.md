### 4.1 댐퍼/팬 자동제어 기본값 (`SystemThresholdTab`)

기준수치 탭 하단 카드. **매장·집진기(ESP) 트리에 있는 집진기마다 1행**이며, REST·DB 연동은 [REST API 설계서 §9.4.3](MetaBeans_ESP_REST_API_엔드포인트_설계서__최신.md)을 따른다.

**테이블**

- 컬럼: 매장·집진기(표시명), 목표 풍량(CMH), 목표 풍속(m/s), 작업
- 각 행 `InputNumber`로 **집진기별 개별** 수정

**일괄 적용 (UI 전용 조작 → 저장 시 한 번에 PUT)**

- 풍량·풍속 입력 필드 + 버튼 **「모든 집진기에 적용」**: 현재 목록의 **모든 행**에 동일 값 반영
- **「전체 시스템 기본값」**: 모든 행을 시스템 시드 값(타입 상수: 850 CMH, 3.5 m/s)으로 되돌림
- 행의 **「기본값으로」**: **해당 집진기만** 시스템 시드로 되돌림

위 조작은 클라이언트에서 배열만 바꾼 뒤, 페이지 상단 **저장**으로 `PUT /system/thresholds`(또는 분리된 댐퍼 기본값 API)에 반영한다.
